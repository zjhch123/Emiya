# Emiya 
`Emiya` 精简, 小巧的、基于webpack的SPA项目脚手架

## 0. Emiya是什么?
一个用于创建基于webpack的SPA项目的脚手架工具。

## 1. Emiya使用说明
### 1. 安装
1. 切换淘宝镜像(不推荐)
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
2. 安装Emiya
```
cnpm i emiya -g / npm i emiya -g
```

### 2. 使用
1. 创建工程
```
emiya my-project
```
2. 安装依赖
```
npm i
```
3. 调试
```
npm run dev
```
4. 打包
```
npm run build
```
